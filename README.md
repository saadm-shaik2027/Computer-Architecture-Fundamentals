# Computer Architecture from First Principles
This project documents my journey through the "Nand2Tetris" curriculum (The Elements of Computing Systems). My goal is to understand the complete abstraction stack by building a 16-bit computer starting from a single NAND gate.

## 🎯 Learning Objectives
* **Hardware:** Designing logic gates, the ALU, and the CPU using HDL.
* **Software:** Building an assembler, a virtual machine, and a compiler.
* **Architecture:** Mastering the Von Neumann architecture and memory mapping.

## 📂 Repository Structure
* `/Daily-Logs`: Detailed notes and "Aha!" moments from each study session.
* `/Hardware-Design`: HDL implementations of logic gates and chips.
* `/Projects`: Completed milestones from the Nand2Tetris curriculum.
