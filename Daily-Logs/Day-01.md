# The Philosophy of Hardware design

**Concept covered:** Abstraction vs Implementation

Today I understood what abstraction and implementation is. Normally I used to wonder, how people code without knowing a single bit of 
knowledge about how a High level code gets converted into binary language which computer understands but today I understood that we dont need 
to understand whats happening underneath to learn coding and this is called abstraction. Whereas the part which is covered is called implemantaion.

**Abstraction:** Abstraction is something that manages complexity and it allows developers use the system without understanding internal 
machines and mechanics.

**Implementation:** It is the underlying code, logic and data structures that executes those functions.
